Bài 4: Giới thiệu HTML cơ bản

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu để tạo cấu trúc trang web.

Các thẻ HTML quan trọng:

  • Thẻ tiêu đề: h1, h2, h3...
  • Thẻ đoạn văn: p
  • Thẻ liên kết: a
  • Thẻ hình ảnh: img
  • Thẻ form: form, input, button
